# Menime

Simple Anime Streaming Web App built with React.js

## Installing

Clone the Repository

```
git clone https://github.com/SatuSembilanDua/Menime
```

Install and Run

```
pnpm install
pnpm run build
pnpm run preview
```

development mode

```
pnpm run dev
```

## Built With

- [React JS](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
